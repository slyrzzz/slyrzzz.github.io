## Resumen

Desarrollo integral de una plataforma de seguridad digital diseñada para mitigar estafas en el comercio electrónico local. El sistema permite a los usuarios verificar la reputación de vendedores y compradores mediante un motor de búsqueda basado en reportes de la comunidad, centralizando información crítica para fomentar transacciones seguras.

## Características principales

- **Motor de Búsqueda de Alta Velocidad:** Implementación de consultas optimizadas para verificar nombres, cédulas o perfiles en milisegundos, permitiendo a los usuarios tomar decisiones informadas antes de una compra.
- **Sistema de Reportes Ciudadano:** Interfaz segura para el registro de incidencias, gestionando una base de datos dinámica de vendedores confiables y usuarios con reportes de malas prácticas.
- **Donaciones en Tiempo Real:** Integración de un sistema de soporte económico con actualizaciones en vivo, permitiendo la sostenibilidad del proyecto mediante la participación activa de la comunidad.
- **Seguridad y Privacidad de Datos:** Arquitectura diseñada para manejar información reportada de manera organizada y protegida, asegurando la integridad de la base de datos.

## Tecnologías utilizadas

- **Frontend & Framework:** Next.js (React) para una interfaz rápida, reactiva y optimizada para SEO.
- **Backend & Database:** Supabase (PostgreSQL) para la gestión de datos relacionales, autenticación y almacenamiento en la nube.
- **Query Language:** SQL para la creación de vistas y funciones de búsqueda complejas y eficientes.
- **Workflow & Deployment:** GitHub para el control de versiones y Vercel para el despliegue continuo (CI/CD).

## Desafíos y Aprendizajes

El desafío técnico principal fue la optimización de las consultas SQL para garantizar que la búsqueda fuera instantánea a medida que la base de datos crecía. Además, la implementación del sistema de donaciones en tiempo real requirió un manejo preciso de los estados del frontend. Aprendí a estructurar bases de datos relacionales escalables y a gestionar la lógica de "crowdsourced data" (datos generados por usuarios) manteniendo la coherencia y velocidad del sitio.

## Resultado

Panacheck se ha convertido en una herramienta de referencia para la comunidad, ayudando a prevenir fraudes y promoviendo la transparencia en el mercado digital. El proyecto demuestra mi capacidad para construir soluciones end-to-end que resuelven problemas sociales reales mediante tecnología avanzada.
