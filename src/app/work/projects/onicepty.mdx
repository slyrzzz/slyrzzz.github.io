## Resumen

Desarrollo integral de la plataforma de comercio electrónico para Ónice, una marca de relojes y accesorios. El proyecto consistió en construir una experiencia de compra premium desde cero, integrando un motor de ventas escalable con un diseño visual minimalista y de alta conversión, gestionando todo el ciclo de vida del producto: desde el desarrollo web hasta la dirección creativa.

## Características principales

- **Arquitectura Headless E-commerce:** Implementación de un frontend desacoplado para garantizar una velocidad de carga superior y una flexibilidad total en la interfaz de usuario.
- **Gestión de Catálogo y Checkout:** Integración de un sistema robusto para la administración de inventario, variantes de producto y un flujo de pago optimizado para dispositivos móviles.
- **Dirección Creativa y Multimedia:** Producción propia de fotografía y videografía de producto de alta resolución, optimizada para web para mantener la calidad visual sin sacrificar el rendimiento (Lighthouse scores altos).
- **Escalabilidad de Marca:** Diseño de una plataforma preparada para el crecimiento, con capacidad de integrar múltiples pasarelas de pago y gestión logística automatizada.

## Tecnologías utilizadas

- **Next.js & TypeScript:** Para una aplicación web ultra rápida, con renderizado estático (SSG) que mejora el SEO y la experiencia del usuario.
- **Saleor:** Como motor de e-commerce (backend), permitiendo una gestión de datos profesional y escalable.
- **Vercel & GitHub:** Pipeline de despliegue continuo (CI/CD) para asegurar actualizaciones constantes y alta disponibilidad.
- **Figma:** Diseño de prototipos de alta fidelidad y sistemas de diseño consistentes.

## Desafíos y Aprendizajes

El mayor reto fue la optimización de recursos multimedia. Al ser una marca de accesorios, la calidad visual es crítica, lo que suele afectar la velocidad. Aprendí a implementar formatos de imagen modernos (WebP/AVIF) y técnicas de lazy loading avanzadas para que el sitio fuera visualmente impactante y técnicamente liviano. Además, manejar el stack de Saleor me permitió profundizar en la comunicación avanzada entre APIs y frontends modernos.

## Resultado

Ónice es el reflejo de mi capacidad para unir el desarrollo técnico con la visión de negocio. El resultado es una tienda online funcional que no solo genera ventas, sino que proyecta una imagen de marca internacional, logrando una retención de usuarios superior a la media de la industria gracias a su rendimiento y diseño.
